Code Next Programs

(At least the ones I am in)

Code Next has many programs that you can join that can lead into other programs.

  • Foundations

    • Code Next Foundations is the beginning of it all. You will have to go to Foundations to get accepted into the next Google programs. Foundations teaches you coding, leadership, and stratagy. It will teach you things that will come in handy if you choose to go into a technology based career. A lot of these tactics can even apply to everyday life.

    These next programs are where most of the tactics you have just learned come into play.

  • Team Edge

    • Team Edge is a program that only the best of Foundations get into. You are supplied with a MacBook from Google. Team Edge takes it to the next level with coding. You learn to code even more and even learn a little animating. You learn to use html, css, and javascript. Literlly, this website is made from the html we are learning. Team Edge has a range of activities that help you in different aspects. If you try hard enough in Foundations, you will surely get into Team Edge.
